## 风险与依赖

- 团队模板需求仍待澄清，模板格式、版本、回滚、覆盖策略和远程仓权限是主要不确定项。
- 依赖 R-0630 的租户管理、团队空间租户归属、知识库文件夹和 Harness 配置解析能力。
- F-005 输入增强依赖 R-0630 已交付的 Agent 任务创建、继续输入、开放式询问回答、Harness 快照和外部 agent core 适配链路。
- US-005-007、US-005-008 需在 R-0730 前确认 Agent 功能目录字段、Harness 知识空间文件引用限制，以及传递给外部 agent core 的字段契约。
- 远程配置仓同步失败、配置解析失败和模板应用部分成功的恢复策略需要在设计评审前明确。
- 全局模板和租户级模板的可见范围、优先级和同名处理规则需要产品确认。
