# TC-006-团队 Agent 会话历史与详情查看

## 基本信息

| 字段 | 内容 |
| --- | --- |
| 测试用例编号 | TC-006 |
| 关联功能 | F-006 |
| 关联版本 | R-0630 |
| 优先级 | 高 |

## 前置条件

- 测试账号具备团队空间创建者、管理员、团队成员和访客四类角色样例。
- 测试团队空间已存在多个 Agent 会话和包含多类事件的会话详情数据。
- 测试数据同时覆盖 `source=manual` 的手工 Agent 会话和 `source=harness_pipeline_node` 的 Harness CICD 流水线 Agent 任务会话。
- 测试团队空间内会话数据覆盖当前用户创建会话、父 Harness CICD 流水线任务由当前用户创建的 Agent 任务会话、当天更新会话、近一周更新会话和超过一页的会话数量。
- 会话事件数据包含用户指令或 Agent 任务输入、执行过程事件、不可展示执行过程事件、`final_output_started` 阶段事件、最终 output、修改文件、询问事件和 plan 进度事件。
- 执行过程事件覆盖调用的协作 Agent、使用的 skill、工具、阅读的文件和编辑的文件。
- 执行过程事件包含可展示的普通文件路径、普通工具参数和普通工具结果。
- 执行过程事件包含命中密钥、凭据、个人隐私或安全策略的敏感片段。
- 前端可访问 Agent 主界面、会话看板、会话详情、会话列表筛选搜索分页、事件流、plan 进度接口和 Harness CICD 流水线父任务链接。

## 测试步骤

| 步骤 | 操作 | 预期结果 |
| --- | --- | --- |
| 1 | 创建者、管理员、团队成员进入 Agent 会话看板 | 可以看到团队空间内 Agent 会话历史 |
| 2 | 访客进入 Agent 会话看板 | 访客可以只读查看会话，不展示输入、改名或处理操作 |
| 3 | 查看会话清单字段 | 清单展示会话名称、来源、创建人、状态和最新更新时间 |
| 4 | 查看手工 Agent 会话来源 | 来源展示为手工创建 |
| 5 | 查看 Harness CICD 流水线 Agent 任务会话来源 | 来源展示为 Harness CICD 流水线 Agent 任务，并展示父 Harness CICD 流水线任务摘要 |
| 6 | 切换全部、我的、今天、近一周筛选项 | 列表分别展示全部会话、当前用户创建的手工会话和父 Harness CICD 流水线任务归属当前用户的 Agent 任务会话、按团队空间时区当天更新会话和近一周更新会话 |
| 7 | 输入会话名称关键词进行模糊搜索 | 列表仅展示会话名称命中关键词的会话 |
| 8 | 组合筛选项和会话名称关键词搜索 | 列表展示同时符合筛选项和关键词的会话，并重置到第一页 |
| 9 | 翻页查看会话历史 | 列表按最新更新时间倒序稳定展示会话，分页信息包含总数、总页数、当前页、每页数量、上一页和下一页状态 |
| 10 | 筛选或搜索无结果 | 会话看板展示无结果状态，不展示为空间暂无会话 |
| 11 | 手工会话创建人修改名称后刷新会话看板 | 清单展示修改后的最新会话名称；按新名称可模糊搜索命中 |
| 12 | 点击手工会话进入详情 | 详情页展示会话名称、来源、创建人、状态和最新更新时间 |
| 13 | 点击 Harness CICD 流水线 Agent 任务会话进入详情 | 详情页展示父 Harness CICD 流水线任务、Run、Agent 任务链接和只读来源提示 |
| 14 | 在 Harness CICD 流水线 Agent 任务会话详情尝试继续、改名、取消或重试 | 不展示相关控件；提示回到 Harness CICD 流水线任务页处理操作 |
| 15 | 查看不同状态的历史会话 | 会话状态按运行中、待输入、完成静止态、异常静止态展示 |
| 16 | 查看完成静止态或异常静止态的历史会话 | 历史会话仍保留在会话看板中 |
| 17 | 查看相对更新时间 | 相对更新时间按分钟级展示，并在可见的会话看板和详情页每 60 秒刷新一次，不展示秒级粒度 |
| 18 | 查看包含输入、执行过程和最终 output 的会话详情 | 详情展示输入、Agent 详细执行过程、最终 output 和修改文件，事件顺序正确 |
| 19 | 查看包含协作 Agent、skill、工具和文件事件的会话详情 | 执行过程以用户友好的摘要展示调用的协作 Agent、使用的 skill、工具、阅读和编辑的文件 |
| 20 | 查看普通文件路径、工具参数和工具结果 | 普通用户可以完整看到可展示执行过程事件中的普通文件路径、普通工具参数和普通工具结果 |
| 21 | 查看包含敏感片段的执行过程事件 | 命中密钥、凭据、个人隐私或安全策略的片段被脱敏，事件仍展示并标识脱敏状态，不影响最终 output 展示 |
| 22 | 查看包含不可展示执行过程内容的会话详情 | 详情不展示原始内容，显示已隐藏状态，不影响最终 output 展示 |
| 23 | 模拟会话运行中新增执行过程事件 | 详情页通过事件流流式追加新增执行过程和进度事件，并更新最新更新时间 |
| 24 | 模拟后端返回 `final_output_started` 阶段事件 | 详情页默认收起详细执行过程，用户可以点击展开查看，并切换到最终 output 生成展示 |
| 25 | 查看最终 output 和修改文件 | 最终输出区展示给用户的最终 output 和修改文件清单；文档详情、diff、接受或拒绝不在本用例执行 |
| 26 | 在主界面打开 Agent plan 与进度弹窗 | 弹窗展示 plan、当前步骤、步骤状态和最新进度；关闭弹窗不影响主界面 |
| 27 | 查看包含询问事件的手工会话详情 | 详情展示询问题干和待输入状态摘要；非创建人只读查看 |
| 28 | 父 Harness CICD 流水线任务不可访问时查看 Agent 任务会话 | 仅展示来源标识和不可访问提示，不泄露父级流水线详情 |
| 29 | 模拟事件流中断 | 系统展示连接中断提示，保留已展示事件，并可重新拉取已保存事件 |
| 30 | 当前空间暂无 Agent 会话 | 会话看板展示空状态 |
