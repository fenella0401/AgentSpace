# TC-006-团队 Agent 会话历史与详情查看

## 基本信息

| 字段 | 内容 |
| --- | --- |
| 测试用例编号 | TC-006 |
| 关联功能 | F-006 |
| 关联版本 | R-0630 |
| 优先级 | 高 |

## 前置条件

- 准备创建者、管理员、团队成员和访客账号。
- 测试数据同时覆盖 `source=manual`、`source=agent_flow_node` 和 `source=event_source` 的 Agent 会话。
- 准备父 AgentFlowTask 可访问和不可访问两类会话。
- 准备 eDevOps FE 来源摘要、事件执行任务和看板链接。
- 准备包含最终输出、修改文件、执行事件为空、事件加载失败和事件流中断的会话。

## 测试步骤

| 步骤 | 操作 | 预期结果 |
| --- | --- | --- |
| 1 | 进入 Agent 会话历史 | 列表展示会话名称、来源、状态、创建人、最新更新时间和摘要 |
| 2 | 切换来源筛选为手工创建 | 仅展示 `source=manual` 会话 |
| 3 | 切换来源筛选为 AgentFlow Agent 任务 | 仅展示 `source=agent_flow_node` 会话 |
| 4 | 切换来源筛选为事件源触发 | 仅展示 `source=event_source` 会话 |
| 5 | 切换全部、我的、今天、近一周筛选项 | 列表按当前用户、团队空间时区和更新时间正确过滤 |
| 6 | 使用关键词搜索会话名称或摘要 | 返回匹配会话并保留分页信息 |
| 7 | 查看手工 Agent 会话详情 | 详情展示输入、执行过程、最终输出、修改文件和可用操作 |
| 8 | 查看 AgentFlow Agent 任务会话详情 | 详情展示父 AgentFlowTask、Run、Agent 任务链接和只读来源提示 |
| 9 | 在 AgentFlow Agent 任务会话详情尝试继续、改名、取消或重试 | 不展示相关控件；提示回到 AgentFlow 任务页处理 |
| 10 | 查看事件源触发会话详情 | 详情展示 eDevOps FE 摘要、事件任务和看板链接 |
| 11 | 事件源 payload 包含敏感字段 | 详情只展示白名单摘要，不泄露敏感字段 |
| 12 | 父 AgentFlowTask 不可访问时查看 Agent 任务会话 | 仅展示来源标识和不可访问提示，不泄露父级详情 |
| 13 | 事件任务不可访问时查看事件源会话 | 仅展示事件源触发来源，不泄露事件任务详情 |
| 14 | 查看执行过程为空的会话 | 展示“暂无执行过程”，不影响最终输出展示 |
| 15 | 模拟事件加载失败 | 保留会话基础信息并提示事件加载失败 |
| 16 | 模拟事件流中断后恢复 | 前端通过详情查询补拉最近事件 |
| 17 | 访客查看会话详情 | 只读展示，不出现继续、回答、改名、文档处理或运行控制控件 |
| 18 | 无权限用户访问会话详情 | 后端拒绝并返回无权限提示 |
