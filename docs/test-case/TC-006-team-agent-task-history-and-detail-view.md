# TC-006-团队 Agent 任务历史与详情查看

## 基本信息

| 字段 | 内容 |
| --- | --- |
| 测试用例编号 | TC-006 |
| 关联功能 | F-006 |
| 关联版本 | R-0630 |
| 优先级 | 高 |

## 前置条件

- 测试账号具备团队空间创建者、管理员、团队成员和访客四类角色样例。
- 测试团队空间已存在多个 Agent 任务和包含多类事件的任务详情数据。
- 测试团队空间内任务数据覆盖当前用户创建任务、当天更新任务、近一周更新任务和超过一页的任务数量。
- 任务事件数据包含用户指令、执行过程事件、不可展示执行过程事件、`final_output_started` 阶段事件、最终 output、修改文件、询问事件和 plan 进度事件。
- 执行过程事件覆盖启动的 subagent、使用的 skill、工具、阅读的文件和编辑的文件。
- 执行过程事件包含可展示的普通文件路径、普通工具参数和普通工具结果。
- 执行过程事件包含命中密钥、凭据、个人隐私或安全策略的敏感片段。
- 前端可访问 Agent 主对话界面、任务看板、任务详情、任务列表筛选搜索分页、事件流和 plan 进度接口。

## 测试步骤

| 步骤 | 操作 | 预期结果 |
| --- | --- | --- |
| 1 | 创建者、管理员、团队成员进入任务历史看板 | 可以看到团队空间内任务历史 |
| 2 | 访客进入任务历史看板 | 访客可以只读查看任务，不展示输入、改名或处理操作 |
| 3 | 查看任务清单字段 | 清单展示对话名称、创建人、状态和最新更新时间 |
| 4 | 切换全部、我的、今天、近一周筛选项 | 列表分别展示全部任务、当前用户创建任务、按团队空间时区当天更新任务和近一周更新任务 |
| 5 | 输入对话名称关键词进行模糊搜索 | 列表仅展示对话名称命中关键词的任务 |
| 6 | 组合筛选项和对话名称关键词搜索 | 列表展示同时符合筛选项和关键词的任务，并重置到第一页 |
| 7 | 翻页查看任务历史 | 列表按最新更新时间倒序稳定展示任务，分页信息包含总数、总页数、当前页、每页数量、上一页和下一页状态 |
| 8 | 筛选或搜索无结果 | 任务看板展示无结果状态，不展示为空间暂无任务 |
| 9 | 创建人修改对话名称后刷新任务历史看板 | 清单展示修改后的最新对话名称；按新名称可模糊搜索命中 |
| 10 | 点击任务进入详情 | 详情页展示对话名称、创建人、状态和最新更新时间 |
| 11 | 查看不同状态的历史任务 | 任务状态按运行中、待输入、完成静止态、异常静止态展示 |
| 12 | 查看完成静止态或异常静止态的历史任务 | 历史任务仍保留在任务看板中 |
| 13 | 查看相对更新时间 | 相对更新时间按分钟级展示，并在可见的任务看板和详情页每 60 秒刷新一次，不展示秒级粒度 |
| 14 | 查看包含用户指令、执行过程和最终 output 的任务详情 | 详情展示用户指令、Agent 详细执行过程、最终 output 和修改文件，事件顺序正确 |
| 15 | 查看包含 subagent、skill、工具和文件事件的任务详情 | 执行过程以用户友好的摘要展示启动的 subagent、使用的 skill、工具、阅读和编辑的文件 |
| 16 | 查看普通文件路径、工具参数和工具结果 | 普通用户可以完整看到可展示执行过程事件中的普通文件路径、普通工具参数和普通工具结果 |
| 17 | 查看包含敏感片段的执行过程事件 | 命中密钥、凭据、个人隐私或安全策略的片段被脱敏，事件仍展示并标识脱敏状态，不影响最终 output 展示 |
| 18 | 查看包含不可展示执行过程内容的任务详情 | 详情不展示原始内容，显示已隐藏状态，不影响最终 output 展示 |
| 19 | 模拟任务运行中新增执行过程事件 | 详情页通过事件流流式追加新增执行过程和进度事件，并更新最新更新时间 |
| 20 | 模拟后端返回 `final_output_started` 阶段事件 | 详情页默认收起详细执行过程，用户可以点击展开查看，并切换到最终 output 生成展示 |
| 21 | 查看最终 output 和修改文件 | 最终输出区展示给用户的最终 output 和修改文件清单；文档详情、diff、接受或拒绝不在本用例执行 |
| 22 | 在主对话界面打开 Agent plan 与进度弹窗 | 弹窗展示 plan、当前步骤、步骤状态和最新进度；关闭弹窗不影响主对话界面 |
| 23 | 查看包含询问事件的任务详情 | 详情展示询问题干和待输入状态摘要；非创建人只读查看 |
| 24 | 模拟事件流中断 | 系统展示连接中断提示，保留已展示事件，并可重新拉取已保存事件 |
| 25 | 当前空间暂无任务 | 任务看板展示空状态 |
