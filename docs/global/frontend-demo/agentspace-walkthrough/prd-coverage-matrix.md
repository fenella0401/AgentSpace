# AgentSpace 全局前端演示 PRD 覆盖矩阵

本矩阵用于串讲和一致性复核。演示只使用浏览器内存模拟数据，不代表生产接口或最终组件拆分。

| 功能 | 演示覆盖 | 验收重点 | 明确不补全 |
| --- | --- | --- | --- |
| F-001 团队空间初始化与 Harness 配置 | 零配置完成页、七中心总览、Workflow 画布、Hook 模拟、历史配置草稿导入、测试/发布/差异操作 | 零配置可直接作业；七类资源状态独立；Workflow 和 Hook 发布前需测试；历史配置导入为草稿 | AGENT.md 上下文预算、市场资源契约、Secret Service 细节 |
| F-002 RBAC 权限管理 | 成员表、角色调整、创建者保护、访客只读、租户管理员无空间角色阻断、权限不足提示 | 创建者不可移除/降级/移交；团队成员不能配置 Harness；访客不能写；后端鉴权为准 | 角色变更实时推送策略 |
| F-003 租户管理与权限治理 | 租户列表、租户详情、启停、授权/撤销租户管理员、租户空间清单、创建空间租户校验、devuc 失败 | 系统管理员治理全局租户；租户管理员只看授权租户；停用租户只读；创建空间前必须校验 tenantId | 租户字段唯一性、格式、devuc 错误码和同步周期 |
| F-004 团队模板配置与应用 | 模块化模板编辑、跳过模板、模板应用分项状态、失败项重试、待澄清提醒 | 模板内容全部可选；Tool 只市场引用；失败项不阻止空间默认 Agent；失败项可单独重试 | 模板版本、复制、回滚、历史保留、整体重新应用、同名排序 |
| F-005 任务创建与运行 | 执行对象选择、知识域选择、Workflow 运行图、条件/并行/子 Workflow/输出节点、人工确认、Hook 决定、R-0730 `/` 与 `@` 预览 | 任务快照记录七类 Harness 版本；Workflow 等待确认可恢复；Hook 可允许/阻止/等待确认；访客和管理面身份不可绕过空间角色创建任务 | 确认超时倒计时和真实事件订阅协议 |
| F-006 任务历史与详情 | 任务看板、筛选、搜索、分页、详情时间线、最终输出、修改文件、plan 弹窗、事件流中断恢复、脱敏提示 | 清单展示名称/创建人/状态/更新时间；筛选搜索重置分页；详情展示过程和输出；事件流中断不清空已有内容 | 真实相对时间每 60 秒刷新实现 |
| F-007 文档变更审阅 | 提案列表、内容查看、差异查看、接受、拒绝、冲突、只读、权限不足提示 | 待接受提案展示摘要；有权限用户看内容和差异；任务创建人可处理；已处理/冲突不再展示提交按钮 | 最终差异组件选型和长文档性能策略 |

## 复核口径

- 功能设计文档内的“验收标准”为主。
- `docs/test-case/` 下已存在测试用例作为补充参考。
- 演示主要覆盖 R-0630；F-005 的 R-0730 输入增强仅作为明确标注的预览。
- 发现 PRD 不明确时，应在右侧“待澄清问题”保留，不以演示行为替代产品决策。
